# Анализатор обертонов колокольных сэмплов

Консольная утилита для анализа одного WAV-файла с записью колокола. Программа определяет спектральные обертоны, выводит их частоту, относительную амплитуду, ближайшую ноту 12-тоновой равномерно темперированной системы и отклонение в центах. Также доступна опциональная визуализация через matplotlib: спектрограмма сегмента затухания и график усреднённого спектра с размеченными обертонами.

## Возможности

- Загрузка моно и стерео WAV-файлов (проверено для 24 бит / 48 кГц)
- Приведение стереозаписи к первому каналу
- Пропуск настраиваемой атаки в начале файла
- Усреднение магнитудного спектра на участке затухания
- Обнаружение спектральных пиков с настраиваемой выраженностью, расстоянием и сглаживанием
- Сопоставление каждого пика с ближайшей нотой и вычисление отклонения в центах
- Опциональная визуализация: спектрограмма + график спектра
- Работа без графического интерфейса для CI/автоматизации
- Поддержка конфигурационных файлов INI с переопределением через CLI
- Двуязычная документация (английский / русский)

## Быстрый старт

```bash
python -m venv venv
venv\Scripts\pip install -r requirements.txt

# Базовый анализ
python analyze_bell.py samples/bell.wav

# Сохранить визуализацию в PNG
python analyze_bell.py samples/bell.wav --plot-save --no-show --peak-count 12
```

## Конфигурация

Скопируйте `analyze_bell.ini.example` в `analyze_bell.ini` и отредактируйте значения. Флаги командной строки всегда переопределяют значения из конфига, а значения конфига переопределяют встроенные значения по умолчанию.

```bash
# Сохранить текущую эффективную конфигурацию для редактирования
python analyze_bell.py --save-config my_config.ini

# Использовать собственный конфиг
python analyze_bell.py samples/bell.wav --config my_config.ini
```

Подробности — в [`docs/config.ru.md`](docs/config.ru.md).

## Устранение неполадок

**Не найдено ни одного пика**
- Уменьшите `--prominence` или `--distance`, чтобы обнаружить тихие или близко расположенные обертоны.
- Проверьте, что `--min-freq` и `--max-freq` охватывают нужный диапазон.

**Слишком длинный пропуск атаки**
- Уменьшите `--attack-skip-ms` или используйте более длинный входной файл.

**PNG пустой / метки перекрываются**
- Используйте `--n-labels`, чтобы ограничить число подписанных пиков.
- Подберите `--spectrum-floor` или `--spec-floor`.

**Ошибка бэкенда matplotlib в headless-окружении**
- Всегда используйте `--no-show` вместе с `--plot-save`.
