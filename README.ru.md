# Анализатор обертонов колокольных сэмплов

Мощный инструмент с удобным **веб-интерфейсом на базе Streamlit** для анализа аудиозаписей колоколов. Приложение находит спектральные обертоны, визуализирует их затухание и позволяет легко экспортировать результаты в PDF или MIDI.

## Возможности

- **Интерактивный веб-интерфейс**: Создан на базе Streamlit для визуализации и настройки параметров в реальном времени.
- **Умный поиск транзиентов**: Автоматически находит точное начало удара колокола с помощью быстрого скользящего RMS-окна.
- **Точный анализ звука**: Поддерживает моно и стерео WAV-файлы, позволяя гибко обрезать первоначальный шум атаки.
- **Поиск пиков**: Находит спектральные пики с настраиваемой выраженностью, расстоянием и сглаживанием.
- **Привязка к 12-TET**: Сопоставляет каждый пик с ближайшей нотой в равномерной темперации и вычисляет отклонение в центах.
- **Удобный экспорт**: Сохраняйте результаты в CSV, MIDI (для синтезаторов) или в виде красивых PDF-отчетов с растровыми спектрограммами.
- **Просмотр осциллограммы**: Наблюдайте в реальном времени, какую именно часть атаки вы обрезаете, прямо в боковой панели.

## Быстрый старт

```bash
python -m venv venv
venv\Scripts\pip install -r requirements.txt

# Запуск веб-интерфейса
python analyze_bell.py
```

Streamlit автоматически откроет анализатор в вашем браузере.

## Форматы экспорта

- **Экспорт в PDF**: Создаёт аккуратный отчет, содержащий таблицу данных, график спектра и сжатую спектрограмму.
- **Экспорт в MIDI**: Преобразует самые громкие обертоны (количество настраивается) в MIDI-файл. Длительность каждой ноты зависит от того, как долго звучит соответствующая частота.
